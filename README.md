# Travel Package Management System

## Overview
The Travel Package Management is a Java Spring Boot application designed to help travel agencies manage their travel packages' itinerary and passengers. The system allows travel agencies to create travel packages, add destinations, specify activities for each destination, and manage passengers' sign-ups for activities.

## Features
- Travel Package Management**: Create, update, and delete travel packages with details such as name, passenger capacity, and itinerary.
- Destination Management: Add, edit, and remove destinations with activities available at each destination.
- Activity Management: Specify activities for each destination, including details such as name, description, cost, and capacity.
- Passenger Management: Enroll passengers for activities in travel packages, handle different types of passengers (standard, gold, premium), and manage their balances.

## Usage
1. Clone the Repository: Clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/yourusername/travel-package-management.git
   ```

2. Import the Project: Import the project into your preferred IDE (e.g., IntelliJ IDEA, Eclipse).

3. Build and Run the Application: Build and run the application using your IDE's build and run commands or use Maven/Gradle commands.

## Technologies Used
- Java
- Spring Boot
- Lombok
- JUnit
- Maven / Gradle
